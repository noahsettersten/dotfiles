## Key mappings

(Using shkd and yabai)

CAPS + vikeys: Tmux focus

alt + vikeys: Window Focus
hyper + h/l: Swap windows
hyper + f: Toggle zoom/fullscreen
shft+alt+h/l: Increase size
alt + enter: Open terminal

shift + alt + f/t: Float / BSP layouts
